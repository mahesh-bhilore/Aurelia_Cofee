# Aurelia Coffee

A premium one-page luxury coffee experience built with Next.js 15, React 19, Tailwind CSS, Framer Motion, GSAP, Lenis, and React Three Fiber.

## Features

- Dark luxury visual language with gold accents
- Full-screen animated hero with a 3D coffee cup experience
- Scroll-triggered storytelling and collection reveals
- Horizontal brewing journey with GSAP ScrollTrigger
- Responsive testimonial carousel and elegant contact area
- Accessible, semantic layout with strong motion and micro-interactions

## Getting started

1. Install dependencies
   ```bash
   npm install
   ```
2. Start the development server
   ```bash
   npm run dev
   ```
3. Open http://localhost:3000

## Production build

```bash
npm run build
```
